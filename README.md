## 기능 요구사항

- [ ] 로또 구입 금액에 해당하는 로또를 발급한다.
    - [x] 로또 구입 금액을 검증한다.
    - [ ] 로또 구입 금액에 해당하는 로또 개수를 계산한다.
    - [ ] 로또 번호를 생성한다.
- [ ] 로또 당첨번호와 보너스 볼을 이용해 당첨 통계를 계산한다.
- [ ] 수익률을 계산한다.
